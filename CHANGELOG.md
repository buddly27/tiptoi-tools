# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com),
and this project adheres to [Semantic Versioning](https://semver.org).

## [0.1.0] - 2026-03-07

### Added

- Initial release
- `info` command to show GME file metadata
- `export` command to extract YAML and media files
- `build` command to create GME from tttool-compatible YAML
- `play` command to play audio by OID or media index
- `scripts` command to browse and search scripts
- `games` command to list game structures
- `oids` command to explore OID relationships
